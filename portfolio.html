Hanzi Quest

Hanzi quest is a site I made to test the user's knowledge of Chinese characters. It shows the user a series of characters and asks them whether or not they know what they mean, starting with common characters and then progressing to rarer ones. After 30 characters, the user is shown their results.

Yuan Yuan Hao Shi

This is a website I created for a Taipei curry restaurant. It's a simple static page built using WordPress, with some minor CSS tweaks.

This was the first real website I made that was destined to go live and actually be used by the public. Therefore, aside from being an introduction to the WordPress platform, this project was also a great chance for me to become familiar with website hosting and domain name registration.
